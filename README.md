## 八字排盘说明

八字，又称四柱，是根据出生年、月、日、时的天干地支组合而成的命理学术语。通过八字可以分析：

- **性格特点** - 根据日主天干判断基本性格
- **五行平衡** - 分析金木水火土的旺衰
- **事业方向** - 适合从事的行业类型
- **感情婚姻** - 感情模式和婚姻特点
